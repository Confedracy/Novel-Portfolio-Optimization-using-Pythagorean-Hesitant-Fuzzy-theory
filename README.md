# Novel-Portfolio-Optimization-using-Pythagorean-Hesitant-Fuzzy-theory
Developed as a part of University Mini-Project. A novel approach to portfolio optimization using advanced extensions of fuzzy sets: Hesitant  Fuzzy Set and Pythagorean Fuzzy Set. Demonstrated stronger performance in terms of the Fuzzy Sharpe Ratio, with improvement of 63% over original
model(Hesitant Fuzzy), proving that accounting for emotional and behavioral factors can enhance traditional finance
methods. An upgradation of Markovitz model.
